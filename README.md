## 📌 About the App

**Sewaan Website** is a modern rental web application built with a powerful and scalable frontend stack. It delivers a seamless, high-performance user experience with beautiful aesthetics, smooth animations, and robust functionality — from browsing listings to managing rentals in real time.

---

## 🚀 Technologies Used

| Category | Technology |
|---|---|
| **Framework** | [React 19](https://react.dev/) + [Vite](https://vitejs.dev/) |
| **Styling** | [Tailwind CSS v4](https://tailwindcss.com/) |
| **Routing** | [React Router v7](https://reactrouter.com/) |
| **State Management** | [Zustand](https://github.com/pmndrs/zustand) |

| **Maps** | [MapLibre GL](https://maplibre.org/) |
| **Charts** | [Recharts](https://recharts.org/) |
| **Notifications** | [Sonner](https://sonner.emilkowal.ski/) |

## 📦 Getting Started

Follow these instructions to set up the project locally on your machine.

### Prerequisites

- Node.js (v18 or higher recommended)
- npm, yarn, or pnpm

### Installation

**1. Clone the repository**

```bash
git clone https://github.com/your-username/sewaan-website-vcom.git
cd sewaan-website-vcom
```

**2. Install dependencies**

```bash
npm install
# or
yarn install
# or
pnpm install
```

**3. Configure environment variables**

Create a `.env` file in the root directory and add your Supabase credentials and any required API keys:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

**4. Start the development server**

```bash
npm run dev
```

The application will be available at `http://localhost:5173`.
